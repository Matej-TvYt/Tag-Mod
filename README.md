# Tag Mod
A multiplayer Minecraft tag minigame for Fabric with three gamemodes: **Normal Tag**, **Freeze Mode**, and **Killer Mode**.

---

## Gamemodes

### Classic Tag
One player is the **Chaser** . Hit a Runner to pass the role, Everyone else runs

### Freeze Mode
One player is the **Freezer** . Freeze runners inside ice cages, Frozen players have **15 seconds** before becoming a new Freezer, Other runners can break the cage to rescue them. Round ends when all runners are frozen

### Killer Mode
One player is the **Killer** . Eliminate runners by hitting them — then they can spectate others, Round ends when all runners are eliminated

---

## Commands
```text
/tagstart -- To Start The Round
/tagstop -- To Stop The Round
/tag config normalMode -- To Enable Classic/Normal Mode
/tag config freezeMode -- To Enable Freeze Mode
/tag config killerMode -- To Enable Killer Mode
```

---

## Requirements
* Minecraft 1.21.1
* Fabric Loader + Fabric API
* 2-3+ players
* **ALL PLAYERS MUST HAVE THE SAME VERSION OF THE MOD**
